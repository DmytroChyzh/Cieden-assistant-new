# Cieden Assistant: ElevenLabs Agent Setup

**Where does "bank assistant" / ApexiBank come from?**  
It comes **only from the ElevenLabs agent** configured in the ElevenLabs dashboard (or via API). Our app does not send any bank text — we send Cieden context in `agent_context`. The agent on ElevenLabs servers still has its **system prompt** set to the old bank/financial persona. Fix it once (see below).

The chat on `/voice-chat` (text and voice) runs through **ElevenLabs Conversational AI**, not CopilotKit. For the assistant to respond as Cieden (rather than a financial bot), the agent's system prompt must use the injected context.

## What’s Already Done in Code

- Cieden context is defined in `src/config/ciedenAgentContext.ts`.
- On every text or voice session start, the app sends the **dynamic variable** `agent_context` with this context to ElevenLabs.
- For ElevenLabs to use it, the agent’s **system prompt** must include a placeholder that gets replaced with this text.

## Option A: Fix via script (recommended)

From the project root, run:

```bash
node scripts/set-elevenlabs-cieden-prompt.mjs
```

This calls the ElevenLabs API and sets the agent's system prompt to use `{{agent_context}}` only (no bank text). Requires `ELEVENLABS_API_KEY` and `NEXT_PUBLIC_ELEVENLABS_AGENT_ID` in `.env.local`. After running, start a **new** chat so the new prompt is used.

## Option B: Fix in the ElevenLabs Dashboard

1. Log in at [ElevenLabs](https://elevenlabs.io) → **Conversational AI** → select the agent whose ID is set in `NEXT_PUBLIC_ELEVENLABS_AGENT_ID`.
2. Open the agent settings (Prompt / System prompt / Instructions).
3. At the **top** of the system prompt, add this line:
   ```text
   {{agent_context}}
   ```
4. You can keep or simplify the rest of the prompt; the main context (who Cieden is, role, style, tools) is already provided via `agent_context`.

**Example system prompt in ElevenLabs:**

```text
{{agent_context}}

Answer in the same language the user writes in. Use the available tools when relevant.
```

After saving, new sessions (text and voice) will receive the full Cieden context, and questions like “What can you do?” or “Tell me about Cieden” will be answered as Cieden, not about accounts or loans.

**If the agent still says "bank assistant" or "banking":** The agent's current system prompt in ElevenLabs is still the old one. Replace the **entire** system prompt with only the example above ({{agent_context}} + one line). Remove any other text that describes a bank or financial assistant. Save and start a new chat.

## No Conflicts With Old Files

- Instructions in `CopilotKitProvider` only apply where chat goes through CopilotKit (not on this page).
- Old financial prompts in the code are not used for `/voice-chat` — replies are generated by the ElevenLabs agent. The only required change is in the ElevenLabs dashboard: add `{{agent_context}}` to the agent’s system prompt.
